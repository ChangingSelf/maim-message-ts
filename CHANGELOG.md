# 更新日志

所有重要的项目更改都将记录在此文件中。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## 版本说明

版本号格式：主版本号.次版本号.修订号

- **主版本号**: 当做了不兼容的 API 修改
- **次版本号**: 当做了向下兼容的功能性新增
- **修订号**: 当做了向下兼容的问题修正
